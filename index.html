Projects

SwishList

As an avid basketball player, I wanted an app enabling me to log my shots so I could track my accuracy at different spots on the court to help focus on areas that needed improvement. Additionally, as a youth basketball coach, I felt a mobile app would be perfect for young players to track their progress as well. And by adding a leaderboard and leveling up features, I surmised the competitive element would also encourage players to take more shots with the goal of improving their accuracy.

SwishList is a responsive basketball shot-tracking app that accomplishes all of that and more. Players can log their workouts and shots using a dynamically-generated basketball court map built using custom CSS. Individual statistics are calculated and presented along with leaderboards in addition to the user's profile and motivational quotes for added inspiration.

SwishList was written in React using JSON Server with a REST API.

Github Repository: github.com/BryanNilsen/Swishlist-React
